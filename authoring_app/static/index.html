Parameters
Each Evidence item entering KV space must satisfy:
1. Express a single medical fact or phenomenon
2. Written as a natural clinical sentence
3. Do not include reasoning, conclusions, or explanations
4. Do not span multiple medical dimensions (symptom / mechanism / test / prognosis)

Example (Bad): "SFTSV感染后患者常出现发热、乏力、血小板减少，并可伴有消化道症状，严重者可进展为多器官功能障碍。"
Example (Good): "SFTSV感染的早期症状常包括发热。"

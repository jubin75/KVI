每一条进入 KV space 的 Evidence，必须满足：
1. 表达一个单一医学事实或现象
2. 以临床语言自然句表达
3. 不包含推理、结论或解释
4. 不跨越多个医学维度（症状 / 机制 / 检查 / 预后）

Example (Bad): "SFTSV感染后患者常出现发热、乏力、血小板减少，并可伴有消化道症状，严重者可进展为多器官功能障碍。"
Example (Good): "SFTSV感染的早期症状常包括发热。"

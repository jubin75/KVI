知识注入KVI（Key Value Injection）

KVI 是用于大模型幻觉定点消除和长期记忆外挂的开源框架。
本控制台提供两条人工闭环：① 人工编制知识（编辑 sentence+meta → 编译 KVBank → 注入调试）；② 从文献编制知识（doc_meta → blocks.evidence → 导入 sentence+meta）。
每条 sentence 是一行 JSON（JSONL）。支持多文件（evidence_001.jsonl 等），便于规模化维护与协作。
每一条进入 KV space 的 Evidence，必须满足：
1️⃣ 表达一个 单一医学事实或现象
2️⃣ 以 临床语言自然句 表达
3️⃣ 不包含推理、结论或解释
4️⃣ 不跨越多个医学维度（症状 / 机制 / 检查 / 预后）

【举例说明】
❌ 不合格（信息密度过高）
“SFTSV感染后患者常出现发热、乏力、血小板减少，并可伴有消化道症状，严重者可进展为多器官功能障碍。”

✅ 合格（语义最小化）
“SFTSV感染的早期症状常包括发热。”
“部分SFTSV感染者会出现乏力。”
“SFTSV感染可导致血小板计数下降。”

知识注入KVI（Key Value Injection）

KVI 是用于大模型幻觉定点消除和长期记忆外挂的开源框架。
本控制台提供两条人工闭环：① 人工编制知识（编辑 sentence+meta → 编译 KVBank → 注入调试）；② 从文献编制知识（doc_meta → blocks.evidence → 导入 sentence+meta）。
每条 sentence 是一行 JSON（JSONL）。支持多文件（evidence_001.jsonl 等），便于规模化维护与协作。

# 目标
在不改变现有检索与 slot 裁决逻辑的前提下，引入**三层知识分层输出**，提升回答的专业性与可读性，同时保持反幻觉能力。

---

## 知识分层定义（必须严格区分）

### Layer 0：Evidence-Bound（硬证据层）
- **来源**：PDF / schema / slot / external KV
- **规则**：
  - 只能陈述 evidence 明确支持的事实
  - 不允许扩写、不允许补充机制细节
  - 不允许使用模型常识或推断
- **用途**：作为“是否允许说”的裁决边界

### Layer 1：Domain Prior（领域共识层）
- **来源**：模型内部的长期医学/生物学共识（教科书级）
- **规则**：
  - 允许在 Layer 0 结论基础上做**稳健解释**
  - 只能使用广泛接受的共识性表述
  - 不要求逐条证据，但必须避免新颖或争议性结论
- **用途**：恢复 base LLM 的专业解释能力

### Layer 2：Speculative / Open（推测层）
- **来源**：模型推断、假说、未完全证实的机制
- **规则**：
  - 必须使用不确定性措辞（如“可能”“尚在研究中”）
  - 必须显式标注为推测性内容
- **用途**：补充深度，不参与事实裁决

---

## 生成与约束策略（给代码用）

1. **生成顺序必须固定**
   - 先生成 Layer 0
   - 再生成 Layer 1
   - 最后（可选）生成 Layer 2

2. **slot gating 仅作用于 Layer 0**
   - slot 未覆盖 → Layer 0 输出“暂无证据支持”
   - 不得影响 Layer 1 / Layer 2 的生成

3. **语言策略**
   - 用户提问为中文 → 所有层输出中文
   - evidence 可为英文，但 Layer 0 需中文概括其含义（非逐句翻译）

4. **输出格式（示例）**
   - 【证据支持的结论】
   - 【领域共识解释】
   - 【推测与研究进展（可选）】

---

## 禁止行为
- 不得将 Layer 1 / Layer 2 内容伪装为证据
- 不得在 Layer 0 中引入模型常识
- 不得混合不同层级的语句

---

## 目标效果
- 事实裁决可靠（Layer 0）
- 解释自然、接近 base LLM（Layer 1）
- 深度可控、风险可关（Layer 2）
